keyword_tags
============

Scripts used to find keyword tags for stack exchange posts.
https://www.kaggle.com/c/facebook-recruiting-iii-keyword-extraction

Summary
-------
For every question in the test set, your submission file should contain
two columns: Id and Tags. Id is the unique identifier of a question in
test.csv. Tags should be a space-delimited list predicted tags. You should
maintain the order of questions.
